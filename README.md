# OUTS - Event Ticketing Platform

Short project overview...

## Features
### Consumer Features
- Browse and search events
- Purchase tickets
- Wishlist management
- QR code tickets

### Host Features
- Create and manage events
- View analytics
- Track ticket sales

### Admin Features
- Approve/reject events
- Manage users
- View revenue statistics

## Tech Stack
- Kotlin
- Jetpack Compose
- Firebase
- Room Database
